### Use
```bash
py server.py
```
- ``http://localhost:8000?url={URL}&ref=[referer]``

### Clone Project
```bash
git clone https://github.com/nghyane/BypassCloudfare.git
```
### Install

```bash
cd BypassCloudfare
pip install -r requirements.txt
```

### pip install
```bash
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
python get-pip.py
```

### For dev
- auto create requirements
```bash 
pip install pipreqs
pipreqs .
```
